About MeetingPoint

MeetingPoint is an event hosted by SJSU students for SJSU Students!
We would love to help connect upcoming graduates make the connections they need to find their career.
At this event we will have organizations like Google, Intel, Microsoft, and Zoom present to talk about career or intern oppurtunties with students.
This is also a great event for not only graduating seniors, but ALL SJSU students to network with other SJSU students and create meaningful relationships.
